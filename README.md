# Netcrowd

Netcrowd is a NetLogo micro-task crowdsourcing platform simulator. This simulator was presented in the 16th International Conference on Information Systems and Technology Management on May 29, 2019, in SÃ£o Paulo, Brazil. Any contribution after the conference is welcome!

# How to Make It Work?

## Using the Platform

1- Download NetLogo from: https://ccl.northwestern.edu/netlogo/
2- Download netcrowd's source code from GitHub
3- Run the source code directly from NetLogo
4- Have fun!

## Using the Web

1- Download netcrowd's source code from GitHub.
2- Go to NetLogo Web: https://netlogoweb.org/.
3- Upload the source code directly to the paltform using you browser.
4- Have fun!
